## Известные ограничения этого пакета

- **Аудит выполнен на снимке репозитория от 12.07.2026**; после любых пересборок пакетов контрольные суммы в каталоге нужно перегенерировать (T-09).
- Интерактив карты (JS-исполнение в браузере) проверялся по статическому HTML и данным, но не кликами в реальном браузере — рекомендуем ручной smoke-тест перед релизом (10 минут: слайдер лет, переключение сценариев/стартовых рядов, тултипы, «район без центра»).
- Точные HTTP-коды живого сайта не фиксировались (ограничения среды); статусы определялись по контенту ответов.
- Белорусские тексты написаны в академической норме; желательна вычитка носителем перед публикацией.
- Цифры в статье и разделах соответствуют текущим версиям данных (v2026.4 / пакеты из каталога). При обновлении данных статью нужно сверить заново — список цифр легко найти поиском по «млн», «тыс.», «%».
- Аудит не заменяет профессиональное демографическое рецензирование содержательных методик (это отдельная, желательная на будущее процедура — приглашение к ней встроено в статью).
