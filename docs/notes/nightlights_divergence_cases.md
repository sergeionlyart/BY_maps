# INF-08: кейсы расхождения свет/статистика (пересчёт на production-данных)

Резидуал D = dlogL − β·dlogP (β — межрайонная эластичность класса). Расхождение — сигнал для проверки, не причинный вывод.

- направление: light_above_statistics (подтверждено пересчётом: True)
- период 2012–2024; зоны: BY-HM, r-minski (агломерация = Минск (BY-HM) + Минский район (r-minski); административный Минск отдельно не интерпретируется)

- качество: vnl_2012_partial_first_year, vnl_2021_step_in_period
- проверить: маятниковая мобильность, ввод жилья, занятость по месту работы, дороги и развязки, площадь застройки, энергопотребление, услуги и логистика; свет и площадь следа раздельно
