128 районов и городов обл. подчинения / 194 города и пгт), страны

priznak_451. Единица одна: 210 «чел.».

Годы в наличии: 1990–2019, 2024, 2025. **2020–2023 отсутствуют**
(Белстат не публиковал миграцию за эти годы). На районном уровне
фактические значения с **1994** (1990–1993 пустые). На районном уровне
живёт только поток «Всего по всем потокам миграции» (разбивки по потокам
и странам — только РБ/области/Минск: API просто не возвращает пустые
комбинации). Сумма районов = области (проверено на 2019, diff=0 по всем
областям; Минск районов depth-2 не имеет).

Файлы (сырые ответы indicatorValuesSearch; точные тела запросов —
belstat_request_log.json):

  все 9 потоков x все страны СНГ;
- belstat_net_city_{1994-1999,2000-2009,2010-2019,2024-2025}.json — бонус:
  194 города/пгт depth-3, сальдо, поток «всего»;

- belstat_migration_dims.json — сплющенные измерения (code/name/depth).

## 2. Eurostat (ВНЖ граждан BY)

- eurostat_migr_resvalid_BY.json — действующие разрешения на 31.12
  (citizen=BY, duration=TOTAL, reason=TOTAL, unit=PER, 2015–2025,
  27 стран ЕС + EU27_2020 + NO/CH/IS). ВНИМАНИЕ: unit=NR из ТЗ неверен,
  у датасета единица PER; reason=TOTAL обязателен, иначе пустой ответ.
- eurostat_migr_resfirst_BY.json — первичные разрешения (те же параметры).
- eurostat_request_urls.txt — точные URL.
- Ключевые точки: EU27_2020 сток 2019 = 133 889, 2024 = 386 834
  (соответствует «386,8 тыс.»); PL 2024 = 251 362; LT 2024 = 58 632.
  2025 есть не у всех стран (PL, LT, IT и др. ещё не сдали).
