# INF-12 «Цена пустеющей карты»: словарь данных

Пакет `urban-overhang` v1.0.0 (2026-07-16). Формулы и параметры — METHODS.md;
допущения — `params/assumptions.yaml`.

**Общие соглашения.** CSV: UTF-8, разделитель — запятая, первая строка —
заголовок; пустая строка в ячейке = «нет данных» (не ноль). Идентификатор
города `city_id` — слаг вида `c-*`, единый по всем файлам пакета и проекту
BY Maps. Служебная строка `city_id = "__national__"` — итог по всей стране
(вне городских масок тоже), используется для долевой нормализации.
Координаты — WGS84, десятичные градусы; площади и морфология — Молльвейде
(ESRI:54009), ячейка 100 × 100 м = 1 га. Темпы (`*_annual`) — лог-годовые,
единица 1/год (×100 ≈ %/год). Эпохи GHSL: 1975, 1980, …, 2020 (шаг 5 лет).
Сценарий морфологии кодируется `t{05|10|20}_c{0|1|2}`: порог доли застройки
5/10/20 % × радиус замыкания 0/100/200 м; основной — `t10_c1`.

## 4. Производные соглашения

- Проценты в текстах пакета — из долей соответствующих полей; «п.п./год» —
  разность лог-годовых темпов ×100.
- «Сокращающийся/стабильный/растущий» — по `pgr_annual` 1990–2020 и полосе
  ±0,001 1/год (`assumptions.yaml → growth.pgr_stability_band`).
- Медианы национальной сводки считаются по панели 92 городов качества A/B
  (ТЗ §4.3: класс C — только карточка и приложение; Фаниполь с метриками
  в агрегаты не входит), если не указано иное.
- Любое число сайта обязано совпадать с `computed_results.json` либо
  выводиться из `story.json`; расхождение — дефект сборки
  (`code/verify.py`).
