# AGENT.md — инструкция для ИИ-агента

Ты работаешь с проверяемым пакетом `nightlights-divergence` v1.0.0:
пересчёт трёх исследовательских кандидатов (H1–H3) расхождения ночной
светимости и статистики населения Беларуси + зональная декомпозиция.

## Быстрый старт

```bash
bash code/run.sh
```

Требуется только Python ≥ 3.10 (stdlib). Скрипт: (1) пересчитывает
`research_candidates.json` и `divergence_decomposition.json` из
завендоренных входов, (2) гоняет инварианты, (3) сверяет метрики с
`checks/expected_results.json`. Любое расхождение — ненулевой код
выхода.

## Что где лежит

- `etl/nightlights_divergence.py` — определения кейсов (`CASES`),
  расчёт метрик (`case_metrics`, обе метрики level|share),
  декомпозиция по 119 зонам (`decomposition`).
- `params/assumptions.json` — β-эластичности по классам зон
  (`model.beta_by_class`), классификация зон, demo-слой.
- `web/public/data/nightlights_v2.json` — гармонизированный ряд
  1992–2024 + модель (вход, не пересчитывается здесь; его
  воспроизводство — пакет `nightlights` v2.1.1).
- `checks/checksums.sha256` — байт-в-байт контроль всех файлов.

## Правила интерпретации (обязательно)

1. Статус кейсов — «кандидат». Формулировки вида «X вызван Y»
   запрещены: причина не установлена, есть только совместимость
   данных с гипотезами.
2. H1 метрикозависим: в уровнях +31,2 %, в долях −21,6 %. Любой вывод
   обязан указывать метрику.
3. H2 считается в долях: продуктовый шаг VNL-2021 внутри периода
   искажает уровни (`qualityFlags: vnl_2021_step_in_period`).
4. Резидуалы 2012 года частично затронуты неполным первым годом VNL
   (`vnl_2012_partial_first_year`).
5. Числа публикуются только при `releaseApproved=true`.

## Внешняя проверка

`etl/nightlights_checks.py` строит `external_checks.json`: направления
открытых рядов против светового резидуала. Правила вердиктов зашиты в
`verdicts()` и продублированы полем `rule` в каждой проверке.
Обновить сырьё: `python -m etl.nightlights_checks_fetch` (дата-портал
Белстата; ЛОВУШКИ: лимит 10 лет/запрос, неиспользуемые классификации
передавать ПУСТЫМИ массивами, агрегат «Промышленность» = код 1820).

## Типовые задачи

- «Проверь H2» → `bash code/run.sh`; смотри
  `web/public/data/nightlights/research_candidates.json` →
  `smolevichi-zhodino.metrics`.
- «Какие ещё зоны расходятся?» → `divergence_decomposition.json`,
  сортируй `rows` по `residualShare` (устойчивее к методпереходам).
- «Пересчитай с другой β» → правь `params/assumptions.json` →
  `model.beta_by_class`, перезапусти run.sh (сверка verify.py при
  этом ожидаемо упадёт — это индикатор отклонения от релиза).
