# Ограничения

1. **Корреляция, не причинность.** Резидуал D = Δlog L − β·Δlog P
   показывает лишь несоответствие света демографической динамике при
   заданной эластичности. Статус всех кейсов — «кандидат»: причина не
   установлена; проверка требует внешней статистики (производство,
   занятость, энергопотребление, миграция, инфраструктура).
2. **Метрикозависимость.** H1 в уровнях +31,2 %, в долях −21,6 %:
   агломерация растёт по абсолютному свету, но теряет долю на фоне
   роста сельского света. Любой вывод обязан указывать метрику.
3. **Методпереходы ряда.** Продуктовый шаг VNL-2021 внутри периода H2
   искажает уровни — H2 считается в долях; первый год VIIRS (2012)
   неполон (`vnl_2012_partial_first_year`).
4. **β фиксирована классом зоны.** Межрайонные эластичности
   (урбан 1,0 / industrial 1,60 / rural 1,66) оценены поперечно и
   перенесены на динамику — это допущение, а не факт.
5. **1992–2011 — реконструкция.** Кейсы считаются только на
   VIIRS-наблюдениях (2012+); DMSP-часть ряда в пакете не
   используется для резидуалов.
6. **Агрегация зон.** Кейсы объединяют по 1–2 зоны; внутризонная
   неоднородность (посёлок против промплощадки) не разрешается.
