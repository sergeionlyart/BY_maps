# Расхождение свет/статистика: пересчёт кандидатов H1–H3 (v1.0.0)

Пакет воспроизводит вычисление трёх исследовательских кандидатов
проекта «Население Беларуси, 1897–2026» — территорий, где ночная
светимость систематически расходится со статистикой населения:

| id | кейс | метрика | период | резидуал света |
|---|---|---|---|---|
| H1 `minsk-agglomeration` | Минская агломерация | уровни | 2012–2024 | **+31,2 %** (доли: −21,6 % — метрикозависимость) |
| H2 `smolevichi-zhodino` | Смолевичи–Жодино | доли | 2019–2024 | **−27,3 %** |
| H3 `astravets` | Островец | доли | 2012–2024 | **+23,1 %** |

Резидуал: `D = Δlog L − β·Δlog P`, где β — межрайонная эластичность
класса зоны (`params/assumptions.json`). Положительный D — свет растёт
сильнее ожидаемого по населению; отрицательный — отстаёт.

**Статус всех кейсов — «кандидат»: данные совместимы с гипотезами,
причина не установлена.** Расхождение не доказывает проблему — оно
указывает, где искать. Гейт публикации: числа показываются только при
`releaseApproved=true` (направление подтверждено пересчётом).

## Воспроизведение

```bash
bash code/run.sh        # пересчёт → инварианты → сверка (stdlib)
```

Входы завендорены: `web/public/data/nightlights_v2.json`
(гармонизированный ряд INF-08 v2.1), `web/public/data/data.json`
(демографический ряд), `params/assumptions.json`. Выход —
`web/public/data/nightlights/research_candidates.json`,
`divergence_decomposition.json` (119 зон, оба резидуала level|share)
и отчёт `docs/notes/nightlights_divergence_cases.md`.

## Состав

- `etl/nightlights_divergence.py` — кейсы, метрики, декомпозиция;
- `etl/nightlights_model.py` — классификация зон и загрузка допущений;
- `checks/tests/test_invariants.py` — знаки, согласованность, гейт,
  запрет причинных формулировок;
- `code/verify.py` — сверка с `checks/expected_results.json`;
- `LIMITATIONS.md` — методпереходы VNL-2012/2021, метрикозависимость H1.
