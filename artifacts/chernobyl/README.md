# Чернобыльский след: проверяемый пакет (INF-07, v1.0.0)

Пакет воспроизводит все числа исследования
«[Чернобыльский след](https://by-population-maps.vercel.app/research/chernobyl)»:
реестр зон радиоактивного загрязнения по официальным НПА (с реквизитами
редакций и завендоренными PDF), классификацию районов, подбор контрольных
районов и сравнение траекторий населения 1970–2026.

## Воспроизведение одной командой

```bash
bash code/run.sh
```

Требования: Python ≥ 3.10, только стандартная библиотека (~2 с).

## Главные результаты

- Класс 1 (зона эвакуации/отселения, ныне Полесский заповедник):
  **Брагинский, Хойникский, Наровлянский** — к 2019 г. сохранили 28–39%
  населения 1979 г. и отстали от контрольных районов на **14–20 п.п.**
  индекса (среднее −16,1 п.п.).
- Класс 2 (сильно загрязнённые, 9 районов с закрытыми отселёнными
  территориями): 7 из 9 отстали от контролей; исключения — Добрушский
  (+13 п.п.) и Костюковичский (+4 п.п.), удержанные райцентрами-городами.
- Реестр зон: суммы НП по районам сходятся с официальными итогами
  перечней **до единицы** (2022 НП в редакции-2021; 2193 — в 2016-й).
- Закрытые отселённые территории 12 районов — 4365 км² (табл. 12 МЧС).

## Структура пакета

- `data/raw/chernobyl/` — официальные PDF обоих перечней (пост. СМ РБ
  № 75/2021 и № 9/2016), текстовый слой брошюры МЧС, агрегат
  `districts_zones.json`;
- `web/public/data/data.json` — ряды населения районов (база проекта);
- `etl/chernobyl.py` — классификация, матчинг, выгрузка;
- `params/assumptions.yaml` — допущения матчинга с обоснованиями;
- `checks/` — контрольные метрики и инварианты; `code/verify.py` — сверка.

Методика, ограничения и инструкции для агентов — `LIMITATIONS.md`,
`PROVENANCE.md`, `AGENT.md`.
