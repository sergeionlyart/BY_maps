# Инструкция для ИИ-агента: аудит пакета INF-07 «Чернобыльский след»

Ты работаешь с проверяемым пакетом исследования. Всё, что заявлено на
лендинге, обязано воспроизводиться отсюда одной командой.

## Быстрый старт

```bash
bash code/run.sh   # Python >= 3.10, стандартная библиотека, ~2 с
```

Ожидаемый финал: `Все 10 контрольных метрик воспроизведены в допусках.`

## Что проверять

1. **Первоисточники**: `data/raw/chernobyl/pravo_75.pdf` и
   `pravo_2016_9.pdf` — официальные PDF с pravo.by (sha256 в
   `sources/registry.csv`). Сами акты — чистые перечни БЕЗ итоговых строк;
   итог «2022 НП» подтверждается табл. 3 брошюры МЧС
   (`belarus_chernobyl_36.txt`, с разбивкой по областям и зонам), итог
   «2193» — там же через дельту −171 к Перечню-2016 и независимым
   построчным пересчётом PDF. Сверь с суммами в
   `data/curated/chernobyl_zones.csv` (инвариант: до единицы).
2. **Классификация**: класс 1 — официальный список эвакуации-1986
   (брошюра МЧС, разд. 4.3.7: Брагинский, Наровлянский, Хойникский);
   класс 2 — районы табл. 12 (закрытые отселённые территории), кроме
   Климовичского (534 га, 0 отселённых НП со строениями — пограничный).
3. **Детерминизм**: повторный прогон даёт байт-в-байт тот же
   `web/public/data/chernobyl.json`.

## Стресс-тесты допущений (params/assumptions.yaml)

Перед каждым перезапуском с правкой кода удаляй байткод-кэш
(`rm -rf etl/__pycache__`), иначе правка, попавшая в ту же секунду mtime,
может молча не примениться.

- **Матчинг**: замени в `etl/chernobyl.py` штрафы (west1921 0,5; чужая
  область 0,15) на 0 и перезапусти. Ожидание: часть контролей сменится,
  но знак эффекта класса 1 (все 3 района отстают) сохранится — это
  инвариант, а не артефакт весов.
- **Базовый год**: индексация к 1970 вместо 1979 — разрывы меняются
  количественно, класс 1 остаётся худшим.
- **Исключение пригородов Минска**: убери `minsk_suburbs` — Хойникский
  получит контролем Дзержинский (агломерационный рост +36% к 1979) и
  разрыв абсурдно вырастет; исключение консервативно (занижает эффект).

## Ловушки

- Матрица «район → число НП» считает НП, а не жителей: район с 43 малыми
  деревнями в зоне может быть менее затронут, чем район с одним городом.
  Для тяжести используй закрытые площади (`closedHa`) и класс.
- Ряды районов между переписями — оценки; резкие изломы вокруг 1986–1991
  сглажены официальной статистикой (реальное отселение было ещё резче).
- Сужение перечней 2016→2021 (−171 НП) — и распад изотопов, и пересмотр
  критериев; не интерпретируй как «улучшение» без оговорок.
