# Лицензии

## Пакет

Код (`etl/`, `code/`, `checks/`), сценарии, документация и итоговые данные
(`data/curated/forecast_v2026_1.csv`, `web/public/data/forecast.json`) —
**CC BY 4.0** со ссылкой на проект «Население Беларуси, 1897–2026» и версию
прогноза (v2026.1).

## Входные данные

- Переписи 2009/2019 и текущие оценки — открытые данные Белстата
  (OLAP census.belstat.gov.by, дата-портал);
- ASFR по областям — «Демографический ежегодник РБ 2019» (Белстат);
- Таблицы дожития и ASFR страны — HMD/HFD через открытое зеркало OWID
  (CC BY 4.0; сами HMD/HFD — по их условиям использования);
- UN WPP 2024 — CC BY 3.0 IGO;
- Wittgenstein Centre v3 — CC BY 4.0.

Полный реестр с лицензией каждого источника — sources/registry.csv.
