# Зарплата × динамика населения: проверяемый пакет (INF-03, v1.0.0)

Пакет воспроизводит все числа исследования
«[Зарплата × динамика](https://by-population-maps.vercel.app/research/wages)»:
зарплатные дифференциалы 118 районов к Минску (2010–2025), биваритную
классификацию 3×3 и регрессионную эластичность десятилетней динамики
населения по дифференциалу — со всеми спецификациями устойчивости.

## Воспроизведение одной командой

```bash
bash code/run.sh
```

Требования: Python ≥ 3.10, только стандартная библиотека (~5 с).
Конвейер стартует с сырых ответов API дата-портала Белстата
(завендорены с sha256): парсинг → деноминация-2016 → дифференциалы →
классификация → регрессии → сверка с `checks/expected_results.json`.

## Главные результаты

- **Население следует за деньгами**: +10% зарплатного дифференциала к
  Минску ассоциированы с **+5,2 п.п.** десятилетней динамики района
  (2015–2025; t = 9,6 классический, 4,8 робастный HC1; R² = 0,46). Знак
  сохраняется во всех спецификациях, робастная значимость — p ≤ 0,05;
  на переписном окне 2009–2019 — +5,5 п.п.
- **Половина силы связи — пригороды Минска**: без четырёх районов
  агломерации эластичность падает до +2,7 п.п. (остаётся значимой).
- **Крайние углы диагонали 3×3** — 43 района из 118: «бедные убывают,
  богатые растут» (вместе с серединой w1p1 — 60).
- **Исключения содержательны**: Солигорский район — единственный с
  зарплатой выше минской (101%, калий), но динамика −6,9% — крупнейший
  отрицательный остаток; крупнейшие положительные остатки — пригороды
  Минска, Бреста и Бобруйска и Островецкий (стройка АЭС).

## Структура пакета

- `data/raw/wages/` — сырые ответы API (зарплата 10218000003, ВРП на душу
  10202100055); реестр с параметрами запросов — `sources/registry.csv`;
- `etl/wages.py` — весь расчёт (без внешних зависимостей, включая OLS);
- `web/public/data/wages.json`, `data/curated/wages.csv` — итоги лендинга;
- `params/assumptions.yaml` — допущения; `checks/` — метрики и инварианты.

Методика, ограничения, инструкции для агентов — LIMITATIONS.md,
PROVENANCE.md, AGENT.md.
