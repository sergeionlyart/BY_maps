# Происхождение данных (ML-challenger, v1.0.0)

Все входы — производные ранее опубликованных проверяемых пакетов проекта.
Пакет ничего не собирает заново: он берёт мишень из движка CCR и признаки
из выходов INF-03/04/05/08 и переписи WP-F1.

```
                     МИШЕНЬ e_i = ln(факт_2026 / CCR_2026)
```

Защита от утечки: из INF-джейсонов НЕ берутся оконно-посчитанные поля
(wageRel/popChange, access popChange/eu2026/euNadir, nightlights
div/lightRatio/popRatio, свет 2020-2023) — только сырые значения ≤2019.
Провенанс каждого признака построчно — `sources/registry.csv`.

Лицензии наследуются от источников (см. реестр); код — по LICENSE.md.
Первичные данные каждого признака описаны в PROVENANCE соответствующего
пакета (by-maps-wages / -access / -nightlights / -migration / -forecast).
