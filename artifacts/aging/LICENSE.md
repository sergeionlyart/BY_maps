# Лицензии

## Пакет

Код (`etl/`, `code/`, `checks/`), документация и итоговые данные
(`data/curated/aging_indicators.csv`, `web/public/data/aging.json`) —
**CC BY 4.0** со ссылкой на проект «Население Беларуси, 1897–2026»
и версию пакета.

## Входные данные

- Переписи 2009/2019 (возрастно-половые структуры районов) — открытые
  данные Белстата (OLAP census.belstat.gov.by);
- ASFR по областям — «Демографический ежегодник РБ 2019» (Белстат);
- Таблицы дожития — HMD через открытое зеркало OWID
  (CC BY 4.0; сама HMD — по её условиям использования).

Полный реестр с лицензией каждого источника — sources/registry.csv.
