# Происхождение данных (INF-02, v1.0.0)

Полный реестр источников с URL, лицензиями и sha256 — `sources/registry.csv`
(унаследован от этапа WP-F1 проекта). Ключевые звенья:

## Возрастно-половые структуры 2009/2019

- Источник: OLAP-куб **F201N** Белстата, census.belstat.gov.by
  (гостевой доступ), однолетние возрасты × пол × район.
- Выгрузка: браузерные fetch-запросы к Saiku REST API (протокол в
  `sources/registry.csv`); агрегация в 5-летние группы — `etl/census_age.py`
  репозитория проекта (в пакет входит готовый результат
  `data/curated/age2009.csv`, `age2019.csv`).
- Контроль: суммы районов = области = страна, до человека
  (pytest-гейт WP-F1 в CI репозитория).

## Входы контрфактной передвижки

- Смертность: однолетние mx, HMD через открытое зеркало OWID
  (catalog.ourworldindata.org, garden/hmd/2025-10-22). **Внимание:
  в `data/curated/mortality.csv` коэффициенты лежат НА 1000 человек**
  (как в зеркале OWID); к долям они приводятся при загрузке —
  `etl/forecast/data.py:mortality_mx` (деление на 1000).
- Рождаемость: областные ASFR 1969–2018 — «Демографический ежегодник
  Республики Беларусь, 2019», табл. 4.10 (PDF Белстата),
  распарсен в `data/curated/fertility_oblast.csv`.
- Движок: CCMPP из пакета by-maps-forecast-v1.0.0 (бэктест 2009→2019:
  ошибка национального итога −0,41%) — файлы `etl/forecast/` идентичны.
