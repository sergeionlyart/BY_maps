# Ночные огни против официальной статистики: проверяемый пакет (INF-08, v1.0.0)

Пакет воспроизводит все числа исследования
«[Ночные огни](https://by-population-maps.vercel.app/research/nightlights)»:
зональную светимость 118 районов и Минска (2015–2023), докризисные
тренды долей и индекс расхождения «свет против официального населения».

## Воспроизведение одной командой

```bash
bash code/run.sh
```

Требования: Python ≥ 3.10, только стандартная библиотека (~2 с).
Конвейер стартует с завендоренной зональной суммы `zonal_light.csv`
(извлечённой из годовых композитов VIIRS один раз через rasterio) —
доли зон в светимости страны, лог-линейный тренд 2015–2019, индекс
расхождения → сверка с `checks/expected_results.json`.

## Данные

Годовые композиты VIIRS **average_masked** (продукт EOG VNL 2.1/2.2,
Colorado School of Mines / Payne Institute) в обработке WorldPop для
Беларуси: вариант `fvf` (без газовых факелов и вулканов), 100 м,
обрезано по стране, 2015–2023 — подлинный спутниковый композит, а не
модель. Исходные растры (~30 МБ/год) не вендорятся: их URL и официальный
sha256 — в `sources/registry.csv`; извлечение зональных сумм —
`etl/nightlights_extract.py` (документация шага, требует rasterio и сами
растры). Вендорится итог — `data/raw/nightlights/zonal_light.csv`.

## Главный результат

- **Расхождение сосредоточено в индустриальных районах**: у
  Смолевичского (Жодино, БелАЗ; доля света ×0,55 при доле населения
  ×0,99), Лидского (×0,63), Солигорского (×0,70), Борисовского (×0,78),
  Гомельского (×0,80), Оршанского (×0,83), Мозырского (×0,85) доля в
  светимости страны растёт медленнее доли в населении — свет отстаёт
  (в абсолюте он даже вырос, но медленнее общенационального); Минск
  держится вровень (индекс ≈ +0,03).
- **Свет ≠ население**: это может значить недоучтённый отток — или
  деиндустриализацию, светодиоды, энергосбережение. Индекс — маркер
  для проверки другими прокси, а не оценка численности.
- **Малые сельские районы шумны** (крохотная база светимости) — надёжны
  крупные, их и ранжирует таблица.

## Структура пакета

- `data/raw/nightlights/zonal_light.csv` — зональная светимость 119 зон
  × 9 лет; реестр с URL и sha256 исходных композитов — `sources/registry.csv`;
- `etl/nightlights.py` — весь расчёт (доли, тренд, индекс; stdlib);
- `etl/nightlights_extract.py` — документация шага извлечения (rasterio);
- `web/public/data/nightlights.json` — итог лендинга;
- `params/assumptions.yaml` — допущения; `checks/` — инварианты, метрики.

Методика, ограничения (полный перечень причин ложных расхождений),
инструкции для агентов — LIMITATIONS.md, PROVENANCE.md, AGENT.md.
