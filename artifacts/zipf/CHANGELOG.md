# Changelog пакета zipf

## v1.0.0 — 2026-07-11

Первый релиз. 12 срезов 1897–2026 (11 переписей + оценка-2026), наклоны
rank-size методом Габэ–Ибрагимова по топ-20/30/50, примация Минска,
26 контрольных метрик с допуском ±0,001 (наклон и примация каждого среза,
SE базовой оценки 2019, число городов в переписи-2019).

Перед публикацией пакет прошёл независимый агентный аудит: все метрики
воспроизведены; по итогам аудита исправлена документированная величина
сдвига GI/OLS (~0,11, а не ~0,05, — меньше половины SE, выводы без
изменений).
