# Инструкция для ИИ-агента: аудит пакета INF-09 «Шоки XX века»

```bash
bash code/run.sh   # Python >= 3.10, стандартная библиотека, ~2 с
```
Ожидаемый финал: `Все 8 контрольных метрик воспроизведены в допусках.`

## Что проверять

1. **Сорсинг**: каждое событие в `data/raw/shocks/events.json` и каждый
   город в `holocaust.json` имеют ≥1 источник (URL). sha256 сырья — в
   `sources/registry.csv`.
2. **Обрыв ВМВ**: национальный ряд 1940 = 9 046 000, 1950 = 7 709 000,
   разница 1 337 000; довоенного уровня достигли лишь в начале 1970-х
   (1970 ещё ниже 1940).
3. **Доли-1897 своим кодом**: доля евреев = идиш / всё население; сверь
   Пинск (~74%), Докшицы (~76%). Кросс-проверь по вероисповеданию
   (`census_relig.json`) — доля близка.
4. **Детерминизм**: два прогона — байт-в-байт одинаковый shocks.json.

## Ловушки

- **Холокост: доля-1897 — НЕ оценка жертв города**; население 1939→1959
  часто восстанавливается иным населением (LIMITATIONS §2) — не
  интерпретируй рост общего населения как «Холокост не затронул».
- **Язык ≠ этничность** — идиш это приближение, не тождество.
- Некоторые города-1897 без project id (нет на карте) — это не дефект.
- Чернобыль-1986 намеренно НЕ здесь (INF-07).
