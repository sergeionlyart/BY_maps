# Демографические шоки XX века: проверяемый пакет (INF-09, v1.0.0)

Пакет воспроизводит числа исследования
«[Шоки XX века](https://by-population-maps.vercel.app/research/shocks)»:
национальный ряд 1897–2026 с событийными вехами (каждая с источником),
оцифровку переписи-1897 по родному языку и данные «города до/после»
Холокоста.

## Воспроизведение одной командой

```bash
bash code/run.sh
```

Требования: Python ≥ 3.10, только стандартная библиотека (~2 с).

## Данные

Национальный ряд — база проекта. События — редакторская хронология с
проверяемыми URL (`data/raw/shocks/events.json`). Перепись-1897 по
родному языку по городам — таблицы Демоскопа (`census_lang.json`,
кросс-проверка по вероисповеданию — `census_relig.json`). Холокост —
доля евреев-1897 + население 1939/1959 с источниками
(`holocaust.json`). Реестр с sha256 — `sources/registry.csv`.

## Главный результат

- **Обрыв Второй мировой**: 9,05 млн (1940) → 7,71 млн (1950),
  −1,34 млн; довоенной численности страна достигла лишь в начале 1970-х.
- **Исчезновение местечка**: в переписи-1897 евреи (идиш) составляли
  55–76% в десятках городов (Докшицы 76%, Пинск 74%, Слоним 72%,
  Слуцк 71%). Холокост 1941–1943 уничтожил это население; города позже
  восстановились иным, нееврейским — местечко исчезло как тип поселения.
  Доля-1897 — мера утраченного, НЕ оценка жертв конкретного города.

## Структура пакета

- `data/raw/shocks/` — перепись-1897 (язык/религия), события, Холокост
  (все с источниками);
- `etl/shocks.py` — сборка ряда, долей, событий (stdlib);
- `web/public/data/shocks.json` — итог лендинга;
- `params/assumptions.yaml` — допущения; `checks/` — 8 метрик.

Методика, ограничения (язык ≠ этничность; композиционная рамка
Холокоста), инструкции — LIMITATIONS.md, PROVENANCE.md, AGENT.md.
