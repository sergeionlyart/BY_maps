# Лицензии

## Пакет

Код (`etl/`, `code/`, `checks/`), документация и итоговые данные
(`web/public/data/migration.json`) — **CC BY 4.0** со ссылкой на проект
«Население Беларуси, 1897–2026» и версию пакета.

## Входные данные

- Сальдо/прибывшие/выбывшие — открытые данные дата-портала Белстата;
- ВНЖ по странам — Eurostat (свободное переиспользование с указанием
  источника);
- Матрица F602 — открытые данные переписи-2019 (census.belstat.gov.by);
- Снапшоты публикаций (zerkalo.io, PAP, nashaniva, belsat, Демоскоп,
  DHS, IOM) — вендорятся в объёме, необходимом для проверки цитируемых
  цифр (исследовательское цитирование); права на тексты у изданий;
- Ряды населения — компиляция pop-stat.mashke.org по официальным
  переписям и оценкам Белстата.

Полный реестр с лицензией каждого файла — `sources/registry.csv`.
