# Пирамида, которая переворачивается (INF-11, v1.0.0)

Национальные возрастно-половые структуры Беларуси **1959–2075**:
128 кадров по 17 пятилетних групп × пол, каждый кадр несёт тип
достоверности.

| тип | кадры | источник |
|---|---|---|
| census | 1959, 1970, 1979, 1989 | переписи БССР (Демоскоп Weekly), до человека |
| census | 2009, 2019 | переписи РБ (OLAP F201N), до человека |
| estimate | 1990–2018, 1999, 2020–2026 | оценки Белстата на 1 января |
| interpolated | 1960–1988 (межпереписные) | когортный варп (визуальная интерполяция) |
| model | 2030–2075 ×3 сценария ×2 старта | экспорт CCMPP v2026.4 |

Контрольные итоги (до человека): 1959 = 8 054 648; 1970 = 9 002 338;
1979 = 9 532 516; 1989 = 10 151 806; 2009 = 9 503 807 (unknown 296
сохранён); 2019 = 9 413 446; 2026 = 9 056 080. Будущее согласовано с
итогами прогноза (±0,1%); WPP-2024 — только кросс-чек (medium-2075 на
4,3% ниже base — сценарии проекта ≠ варианты ООН).

## Воспроизведение

```bash
bash code/run.sh   # сборка -> инварианты -> WPP-кросс-чек -> сверка
```

Только stdlib. Свежая добыча сырья — `python -m etl.pyramids_fetch`
(Демоскоп + дата-портал Белстата; вендоренные копии с sha256 — в
`sources/registry.csv`).

Лендинг: <https://by-population-maps.vercel.app/pyramid> — морфинг,
«найди себя» (`?born=1985&year=2050&scenario=base`), аннотации A1–A7.
