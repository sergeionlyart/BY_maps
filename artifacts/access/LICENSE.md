# Лицензии

## Пакет

Код (`etl/`, `code/`, `checks/`), документация и итоговые данные
(`data/curated/travel_times.csv`, `web/public/data/access.json`) —
**CC BY 4.0** со ссылкой на проект «Население Беларуси, 1897–2026»
и версию пакета.

## Входные данные

- Дорожный граф (`data/raw/osm/graph_edges.csv.gz`) — производная
  работа от OpenStreetMap: **ODbL 1.0**, © участники OpenStreetMap
  (выгрузка Geofabrik); при переиспользовании графа действуют условия
  ODbL (share-alike для производных баз данных);
- Реестр погранпереходов — компиляция проекта по официальным
  публикациям (granica.gov.pl, MSWiA, VSAT, ГПК РБ), CC BY 4.0;
- Зарплаты — открытые данные дата-портала Белстата;
- Ряды населения районов — компиляция pop-stat.mashke.org по официальным
  переписям и оценкам Белстата (использование с указанием источника).

Полный реестр с лицензией каждого источника — `sources/registry.csv`
и `sources/registry_border.csv`.
