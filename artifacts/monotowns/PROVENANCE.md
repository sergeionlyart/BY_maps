# Происхождение данных (INF-06, v1.0.0)

```
Открытые источники (ручная курация):
  сайты предприятий · отраслевая пресса · годовые отчёты ·
  санкционные списки EU (consilium.europa.eu, eur-lex) ·
  US (OFAC/Treasury) · UK · Canada
        │  (построчные URL в каждой записи)
        ▼
  data/raw/monotowns/registry.json  (49 пар город-предприятие:
  отрасль, год, занятость-оценка, моно-зависимость, санкции[дата],
  sources[url]) -> плоский data/curated/monotowns.csv
        │
        ▼  etl/monotowns.py (стандартная библиотека):
  типология по отрасли; полоса риска = зависимость + санкции;
  индекс населения к 1989=100; matched comparison (8 ближайших по
  населению-1989 не-моногородов/не-облцентров -> медианный контроль)
        │  ряды городов: web/public/data/data.json
        ▼
  web/public/data/monotowns.json
```

Все шаги от `registry.json` детерминированы; сетевых вызовов в run.sh
нет. Реестр — компиляция проекта (CC BY 4.0); первичные факты — по своим
источникам (URL в registry.json/monotowns.csv), санкции — с юрисдикцией
и датой. Занятость и моно-зависимость — экспертные оценки по источникам,
актуальны на 2026-07-12.
