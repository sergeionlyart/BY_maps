# Лицензии

## Пакет

Код (`etl/`, `code/`, `checks/`), документация и итоговые данные
(`data/curated/monotowns.csv`, `web/public/data/monotowns.json`) —
**CC BY 4.0** со ссылкой на проект «Население Беларуси, 1897–2026» и
версию пакета.

## Входные данные

- Реестр моногородов — компиляция проекта (CC BY 4.0); первичные факты
  (занятость, санкции, история предприятий) — по своим источникам,
  URL которых указаны построчно в `data/raw/monotowns/registry.json` и
  `data/curated/monotowns.csv` (сайты предприятий, отраслевая пресса,
  санкционные списки EU/US/UK/Canada — их условия использования);
- Ряды населения городов — компиляция pop-stat.mashke.org по официальным
  переписям и оценкам Белстата.

Реестр источников с sha256 — `sources/registry.csv`.
