# Зарплата × динамика населения: данные и методика

## 1. Данные и источники

Номинальная начисленная среднемесячная заработная плата по 118 районам,
10 городам областного подчинения, областям и стране, 2010–2025 —
открытый API дата-портала Белстата (индикатор 10218000003, годовая
периодичность; сырые ответы завендорены с sha256); численность занятых
(10102000017) и ВРП на душу по областям (10202100055) — тот же портал.
Ряды населения районов — база проекта. Реестр —
`data/raw/wages/registry.csv`.

## 2. Период и актуальность

Основное окно: динамика населения 2015–2025, средний зарплатный
дифференциал 2015–2024. Окно устойчивости: перепись-к-переписи 2009–2019
(дифференциал 2010–2019). Дата обращения: 2026-07-11. Версия v1.0.0.

## 3. Преобразования

Дифференциал района = зарплата района / зарплата г. Минска того же года:
отношение самонормируется — деноминация-2016 и дефлятор не нужны
(в абсолютных рядах старые рубли 2010–2015 делятся на 10 000).
Периметр зарплатной статистики — переписной (районы без городов
областного подчинения; Орша и Полоцк внутри районов) — население района
приводится к нему вычитанием города-хоста. Биваритная классификация:
терцили среднего дифференциала × терцили десятилетней динамики (3×3).

## 4. Модели и допущения

Срезовая регрессия по 118 районам: изменение населения за десятилетие
(п.п.) ~ ln(средний дифференциал) + ln(население райцентра) + const.
Оценка — OLS (чистый Python, нормальные уравнения; код в пакете).
Спецификации устойчивости: без контроля размера центра; окно 2009–2019;
без четырёх пригородов Минска; WLS с весом по размеру центра.

## 5. Валидация

Pytest-гейты в CI: покрытие 118 районов полными рядами 2010–2025;
контрольные значения страны (2024 = 2288,6; 2025 = 2679,7 BYN —
официальные публикации); деноминация (2010 ≈ 121,7 BYN в новых);
дифференциалы в коридоре 0,3–1,1; терцили по ~39–40 районов;
регрессия воспроизводится из пакета одной командой; знак — во всех
спецификациях, значимость t > 3 при классических SE везде, при робастных
к гетероскедастичности (HC1; она сильная, Бройш–Паган p ≪ 0,001) —
t = 4,8 в основной и 4,1 в переписной спецификациях, t = 2,0–2,6 в
консервативных (p ≤ 0,05).

## 6. Сильные стороны

Полное покрытие районов шестнадцатью годами официальных зарплат;
относительный дифференциал устраняет инфляцию и деноминацию по
построению; результат подтверждается на независимом переписном окне
2009–2019; выбросы регрессии совпадают с известными кейсами
(Солигорск — калий, Островец — АЭС, пригороды Минска).

## 7. Ограничения

Зарплата учитывается по месту работы, а население — по месту жительства:
маятниковая занятость (особенно вокруг Минска) завышает дифференциал
пригородов и часть их «эффекта». Связь корреляционная, не причинная:
высокая зарплата и приток населения могут иметь общую причину
(индустриальный якорь, административный статус). Средняя зарплата
маскирует распределение (моногород с одним высокооплачиваемым
предприятием). Занятость по районам обрывается на 2024, ВРП — только
областной. Контроль транспортной доступности появится с INF-04.

## 8. Устойчивость

Эластичность положительна во всех спецификациях: +10% дифференциала
ассоциированы с +2,7…+5,5 п.п. десятилетней динамики (основная оценка
+5,2, HC1 t = 4,8; без пригородов Минска +2,7 — вдвое слабее, HC1
t = 2,6: половина силы связи — агломерационный эффект; на переписном
окне 2009–2019 +5,5, HC1 t = 4,1). R² основной спецификации 0,46.
Знак не меняется ни в одной проверенной вариации; величина зависит от
включения пригородов; пригороды Бреста и Бобруйска (крупные
положительные остатки) из консервативной спецификации не исключались —
агломерационная поправка, вероятно, недооценена.
