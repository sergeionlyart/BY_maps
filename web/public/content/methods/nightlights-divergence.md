# Расхождение свет/статистика: данные и методика

Кейсы H1–H3 (Минская агломерация, Смолевичи–Жодино, Островец) —
расширение исследования «Беларусь из космоса»: полная методика ряда
светимости, резидуалов D = Δlog L − β·Δlog P, гейта releaseApproved и
внешней проверки открытой статистикой описана в §9 методблока
[nightlights](/methodology) и в пакете
[by-maps-nightlights-divergence-v1.0.0.zip](/artifacts/by-maps-nightlights-divergence-v1.0.0.zip)
(README, LIMITATIONS, AGENT.md).

Кратко: резидуал расхождения считается per-case в уровнях либо долях
национального света (продуктовый шаг VNL-2021 исключает уровни для
H2), β — классовая межрайонная эластичность; статус всех кейсов —
«кандидат»: данные совместимы с гипотезами, причина не установлена.
Внешняя проверка (16 проверок: индекс промпроизводства районов
2016–2021, занятость, зарплата, миграция, электроэнергия по областям
до 2020) даёт вердикты consistent/inconsistent/context о
СОГЛАСОВАННОСТИ рядов — не о причине. Воспроизведение: `bash
code/run.sh` в пакете (stdlib, 15 контрольных метрик, байт-в-байт).
